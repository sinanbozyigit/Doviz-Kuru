# Döviz Kuru Uygulaması

React ile yapılmış basit bir döviz kuru uygulaması. 

## Özellikler
- Güncel döviz kurları ile çeviri
- Kullanıcı seçimine göre From/To currency
- Bayrak simgeleri ile görsel destek
- Sonuç input'u sadece okunabilir
